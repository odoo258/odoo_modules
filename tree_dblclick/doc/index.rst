===================
 Tree double click
===================

Double click to open records in tree view instead of one click.
Useful to copy text from tree view to clipboard.
Works for all tree views.

Install
=======

Just install module

How to use
==========

* Open any tree view.
* Click - nothing. Double click - open record.
